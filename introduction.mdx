<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with ImgTool in minutes
  </p>

  <Steps>
    <Step title="Build the project">
      Clone the repository and build using CMake:
      
      ```bash
      git clone https://github.com/100495954/imgtool.git
      cd imgtool
      cmake -S . -B build-release -DCMAKE_BUILD_TYPE=Release
      cmake --build build-release
      ```
    </Step>

    <Step title="Run your first command">
      Compress a PPM image using the AOS implementation:
      
      ```bash
      ./build-release/imtool-aos/imtool-aos input/image.ppm output/compressed.cppm compress
      ```
      
      Or use the SOA implementation for comparison:
      
      ```bash
      ./build-release/imtool-soa/imtool-soa input/image.ppm output/compressed.cppm compress
      ```
    </Step>

    <Step title="Explore other operations">
      ImgTool supports multiple image processing operations. Resize an image:
      
      ```bash
      ./build-release/imtool-aos/imtool-aos input/image.ppm output/resized.ppm resize 800 600
      ```
      
      <Note>
        Both `imtool-aos` and `imtool-soa` variants support the same operations with identical outputs, allowing for direct performance comparison.
      </Note>
    </Step>
  </Steps>
</div>
