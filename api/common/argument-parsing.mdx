### args_checker

Validates command-line arguments based on the operation type and ensures the correct number of parameters.

```cpp
bool args_checker(std::vector<std::string> const &args)
```

<ParamField path="args" type="std::vector<std::string> const&" required>
  Vector of command-line arguments to validate
</ParamField>

<ResponseField name="return" type="bool">
  Returns `true` if arguments are valid for the specified operation, `false` otherwise
</ResponseField>

**Location**: `common/progargs.cpp:14`

#### Operation Requirements

The function validates arguments based on the operation (args[3]):

<ResponseField name="compress" type="4 parameters">
  Compression operation requires exactly 4 arguments
</ResponseField>

<ResponseField name="resize" type="6 parameters">
  Resize operation requires exactly 6 arguments (includes width and height)
</ResponseField>

<ResponseField name="maxlevel" type="5 parameters">
  Max level adjustment requires exactly 5 arguments (includes new max value)
</ResponseField>

<ResponseField name="cutfreq" type="5 parameters">
  Frequency cutoff operation requires exactly 5 arguments (includes cutoff value)
</ResponseField>

#### Error Messages

The function outputs specific error messages:
- "Error : Wrong number of arguments; Needed X parameters"
- "Wrong operation introduced; [operation] is not a valid operation"

### pertenencia

Checks if a given operation name is valid and supported.

```cpp
bool pertenencia(std::string const & operation)
```

<ParamField path="operation" type="std::string const&" required>
  Operation name to validate
</ParamField>

<ResponseField name="return" type="bool">
  Returns `true` if the operation is in the list of valid operations, `false` otherwise
</ResponseField>

**Location**: `common/progargs.cpp:51`

#### Valid Operations

The function recognizes the following operations:

<ResponseField name="compress" type="string">
  Image compression operation
</ResponseField>

<ResponseField name="resize" type="string">
  Image resizing operation
</ResponseField>

<ResponseField name="cutfreq" type="string">
  Frequency cutoff operation
</ResponseField>

<ResponseField name="maxlevel" type="string">
  Maximum color level adjustment
</ResponseField>

<ResponseField name="info" type="string">
  Display image information
</ResponseField>

#### Example

```cpp

    std::cout << "compress is a valid operation" << std::endl;
}

    std::cerr << "invalid is not a supported operation" << std::endl;
}
```

## Complete Usage Example

<CodeGroup>
```cpp Basic argument parsing
#include "common/progargs.hpp"
#include <vector>
#include <string>
#include <iostream>

int main(int argc, char* argv[]) {
    // Check minimum argument count
    if (!progargsCommon::check_argc(argc)) {
        std::cerr << "Usage: imgtool <input> <output> <operation> [params...]" << std::endl;
        return 1;
    }

    // Convert to vector for easier handling
    std::vector<std::string> args(argv, argv + argc);

    // Validate arguments
    if (!progargsCommon::args_checker(args)) {
        return 1;
    }

    // Extract file parameters
    progargsCommon::parameters_files params;
    params.input_file = args[1];
    params.output_file = args[2];
    std::string operation = args[3];

    std::cout << "Input: " << params.input_file << std::endl;
    std::cout << "Output: " << params.output_file << std::endl;
    std::cout << "Operation: " << operation << std::endl;

    return 0;
}
```

```cpp With operation validation
#include "common/progargs.hpp"
#include <vector>
#include <string>
#include <iostream>

int main(int argc, char* argv[]) {
    if (!progargsCommon::check_argc(argc)) {
        return 1;
    }

    std::vector<std::string> args(argv, argv + argc);
    std::string operation = args[3];

    // Check if operation is valid
    if (!progargsCommon::pertenencia(operation)) {
        std::cerr << "Invalid operation: " << operation << std::endl;
        std::cerr << "Valid operations: compress, resize, cutfreq, maxlevel, info" << std::endl;
        return 1;
    }

    // Validate full argument list
    if (!progargsCommon::args_checker(args)) {
        return 1;
    }

    // Process operation
    progargsCommon::parameters_files params;
    params.input_file = args[1];
    params.output_file = args[2];

    return 0;
}
```
</CodeGroup>
